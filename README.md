# House Price Prediction using Linear Regression

## Overview
This project was developed as part of the Machine Learning Internship at SkillCraft Technology. The main objective of this project is to predict house prices using Linear Regression based on important housing features such as living area, number of bedrooms, and bathrooms.

## Dataset
Dataset used:
House Prices - Advanced Regression Techniques

Source:
https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Features Used
- GrLivArea
- BedroomAbvGr
- FullBath

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Feature Selection
4. Train-Test Split
5. Model Training using Linear Regression
6. Model Evaluation
7. Data Visualization

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

## Results
The Linear Regression model was successfully trained and evaluated on the housing dataset. The project demonstrates the complete workflow of a basic machine learning regression problem.

## Conclusion
This project helped in understanding the practical implementation of Machine Learning techniques including data preprocessing, model training, evaluation, and visualization.
